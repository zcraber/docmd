# @docmd/themes

Official themes for **docmd**.

## Included Themes
- **Sky:** Clean, blue accents, modern (Default).
- **Ruby:** Elegant serif fonts, red accents.
- **Retro:** Neon, terminal-style, high contrast.

## Usage
Set the theme in your `docmd.config.js`:

```javascript
module.exports = {
  theme: {
    name: 'retro' 
  }
}
```
